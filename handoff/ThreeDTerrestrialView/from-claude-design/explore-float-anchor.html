浮遊 3D アンカー — デザイン探索 ThreeDTerrestrialView · main / sub
角度をつけたマップ上に 3D オブジェクトを立体表示する時のアンカー。地表に直置きするとマップの視認性が落ちるので浮かせたい。ただ浮かせると違和感が出るため、「この地点にある」と一目で分かる接地表現を探る。main はセグメント／他ユーザーアイコン／サムネを出さない方針。下記 5 方向。
A浮遊感：地表から離して立体を主役に
B接地の明示：地点が曖昧にならない
C非干渉：マップの読みを邪魔しない
TILTED MAP · pitch ~55°

01
ビーコン（接地リング＋光の柱）
地表に発光リングで地点を打ち、細く立ち上がる光の柱で浮いた3Dへ繋ぐ。最も浮遊感が高く、active 時はリングをカラフルに。
接地 = 発光リング / 柱長 = ポール長に連動

3D
02
ホバーステージ（ガラス円盤＋テザー）
3Dを透明ガラスの円盤に乗せ、細いテザーで接地ドットへ降ろす。円盤＝置き場、ドット＝地点。円盤型の正常進化。
接地 = ドット＋リング / テザー長 = ポール長
TILTED MAP · pitch ~55°
3D
03
ライトコーン（光の逆円錐）
地点の一点へ収束する光の逆円錐。クレイの円錐ではなく光で表現するので軽く、浮遊と接地を同時に示す。円錐型の正常進化。
接地 = 収束点 / 円錐高 = ポール長
TILTED MAP · pitch ~55°
3D
04
フロートピン（地点マーカー＋点線）
マップ慣れしたピンを地点に置き、点線ライザーと落ち影で上の3Dに繋ぐ。地点の意味が最も明確。ピンは active でカラフル。
接地 = ピン / 点線長 = ポール長
TILTED MAP · pitch ~55°
3D
05
シャドウアンカー（棒なし・落ち影）
棒で繋がず、真下の落ち影と中心ドットだけで地点を示す。最小・最も上品。浮遊高で影の大小が決まる（軽くゆれる）。
接地 = 影＋ドット / 影サイズ = 浮遊高
